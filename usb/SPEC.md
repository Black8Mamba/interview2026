# USB 学习资料与面试指南 - 设计规格

## 1. 项目概述

**项目名称**: USB 协议与开发面试指南
**项目类型**: 技术文档/学习资料
**核心功能**: 面向中高级嵌入式开发者的 USB 协议栈学习资料，兼顾面试准备

## 4. 内容要点（各章节）

### 4.1 USB 协议基础
- USB 协议分层架构（物理层/协议层/事务层/传输层）
- 四种传输类型：控制/等时/中断/批量
- 端点类型与特性
- 包类型：Token/Data/Handshake/Special
- 同步机制与帧/微帧

### 4.2 USB 枚举过程
- 设备插入检测与复位
- 地址分配流程
- 描述符体系：设备/配置/字符串/接口/端点
- 标准描述符 vs 类特定描述符
- 配置选择与Alternate Setting
- 枚举失败常见原因

### 4.3 USB 类驱动
- HID 类：报告描述符、 boot protocol
- CDC 类：虚拟串口模型
- Mass Storage 类：CBW/CSW 协议
- DFU 类：固件升级流程
- 类驱动选择原则

### 4.4 USB 驱动开发
- 设备端固件开发要点
- 主机端驱动架构
- 复合设备与接口关联
- 驱动程序框架（Linux/Windows/RTOS）
- 常见驱动模型对比

### 4.5 USB 调试与排查
- 工具链：逻辑分析仪、协议分析仪、Bus Hound
- 软件工具：USBView、WireShark、调试日志
- 枚举失败排查思路
- 传输异常分析
- 性能问题定位

### 4.6 USB 高速/超高速
- 高速 vs 全速差异
- 事务调度机制
- 带宽分配与预留
- 协议效率优化
- 常见兼容性问题

### 4.7 OTG 与 Type-C
- OTG 协议：ID 引脚、角色切换
- Type-C 接口特性
- CC 通道与配置
- PD 协议基础
- 常见问题与解决方案
