# Per-CPU 变量

## 概述

Per-CPU 变量是 Linux 内核为每个 CPU 创建一个独立副本的机制。这种设计可以完全避免锁的使用，因为每个 CPU 只访问自己的数据副本，不存在竞争。

## 主要优势

1. **无锁**：每个 CPU 访问自己的副本，无需加锁
2. **缓存友好**：数据在 CPU 本地缓存中，减少缓存争用
3. **无竞争**：CPU 之间不存在数据竞争
4. **简化代码**：无需考虑并发问题
