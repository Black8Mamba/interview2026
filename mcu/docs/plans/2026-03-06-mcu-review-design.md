- 资深工程师（3年以上，高级/专家岗位）

## 4. 知识点详细大纲

### 第一部分：核心基础知识

#### 1.1 ARM Cortex-M内核架构
- Cortex-M0/M0+/M3/M4/M7/M23/M33 架构差异
- 寄存器组（通用寄存器、特殊寄存器）
- 运行模式（Handler模式、Thread模式）
- 特权等级（特权级、用户级）
- NVIC中断控制器
- SysTick定时器
- MPU内存保护单元
- FPU浮点单元 / DSP指令集
- 架构版本（v6-M, v7-M, v8-M）
- TrustZone安全扩展

#### 1.2 汇编语言与寄存器
- Thumb/Thumb-2指令集
- 常用汇编指令（数据传送、算术、逻辑、跳转、比较）
- MRS/MSR特殊寄存器访问
- CPSIE/CPSID开关中断
- DMB/DSB/ISB内存屏障指令
- 汇编与C混合编程 / 内联汇编

#### 1.3 启动流程与内存布局
- Vector table向量表结构
- 启动模式（Flash/System Memory/RAM）
- .isr_vector / .data / .bss 段
- SystemInit() 与启动文件
- 内存映射（Code/SRAM/Peripheral空间）
- 链接脚本（Linker Script）

### 第二部分：外设与驱动

#### 2.1 低速外设

**2.1.1 GPIO/EXTI**
- 输入/输出模式（浮空/上拉/下拉/推挽/开漏）
- 复用功能与重映射
- EXTI触发方式（上升沿/下降沿/双边沿/电平）
- EXTI事件与中断的区别

**2.1.2 复位与时钟(RCC)**
- 复位类型（Power/Soft/WDG/IWDG/LOCKUP）
- 时钟源（HSE/HSI/LSI/LSE/PLL）
- AHB/APB总线时钟
- 时钟安全系统(CSS)/时钟输出(MCO)

**2.1.3 看门狗**
- IWDG独立看门狗：分频器/重载值
- WWDG窗口看门狗：窗口时间/提前唤醒

**2.1.4 RTC**
- 时钟源配置
- 闹钟/周期性唤醒
- 时间戳/备份寄存器

**2.1.5 电源管理(PWR)**
- 调压器(PWR_REGLATER)
- PVD可编程电压检测
- BOR欠压复位

**2.1.6 低功耗模式**
- Sleep模式
- Stop模式
- Standby模式
- 各级模式对比与唤醒方式

#### 2.2 高速外设

**2.2.1 定时器**
- 基本定时器(TIM6/7)
- 通用定时器(TIM2/3/4/5)
- 高级定时器(TIM1/8)
- 预分频器/自动重装载
- 计数模式（向上/向下/中央对齐）
- 输入捕获/输出比较/PWM
- 死区时间/刹车输入
- 定时器同步/编码器接口

**2.2.2 DMA控制器**
- DMA通道/请求映射
- 仲裁器/优先级
- 传输模式（普通/循环/双缓冲）
- 数据宽度/地址增量
- 半满/全满中断

**2.2.3 ADC/DAC**
- SAR型ADC原理
- 采样时间/转换时间
- 扫描模式/连续转换
- 注入通道/常规通道
- ADC校准/DMA请求
- DAC输出/触发源

**2.2.4 通信外设**
- USART/UART: 异步通信/波特率/流控
- SPI: 四种模式/时钟极性/相位
- I2C: 起始停止/ACK/SMBus
- CAN: 位时序/滤波器
- USB: 端点类型/传输模式
- SDIO: 命令/数据传输

**2.2.5 高速通信与存储**
- ETH以太网: MAC/DMA描述符
- DCMI摄像头接口
- FSMC外部存储器接口
- QuadSPI/FMC

**2.2.6 显示接口**
- LTDC: 层配置/时序参数
- DSI: 命令模式/视频模式

#### 2.3 特殊外设

**2.3.1 CRC**
- CRC32/CRC16多项式

**2.3.2 安全加密**
- AES: ECB/CBC/CTR/GCM模式
- DES/TDES
- 加密/解密流程

**2.3.3 随机数**
- RNG真随机数
- PRNG伪随机数

**2.3.4 外部存储扩展**
- FSMC/EMIF
- SRAM/NOR/NAND接口
