|----------|--------|----------|------|--------|------|

## 3. 命令执行流程

### 3.1 典型读取流程

```
1. 发送读命令 (0x03/0x0B/0xEB)
2. 发送24位地址 (A23-A0)
3. [可选] 发送dummy字节
4. 读取数据 (连续读取)
5. [可选] 拉高CS#结束
```

### 3.2 典型编程流程

```
1. 发送写使能命令 (0x06)
2. 读取状态寄存器确认WEL=1
3. 发送页编程命令 (0x02)
4. 发送24位地址
5. 发送数据 (最多256字节)
6. 等待编程完成 (轮询BUSY位)
7. 验证数据
```

### 3.3 典型擦除流程

```
1. 发送写使能命令 (0x06)
2. 发送扇区擦除命令 (0x20)
3. 发送24位地址
4. 等待擦除完成 (轮询BUSY位)
5. 验证擦除 (读取数据应为0xFF)
```

### 4.2 常用命令组合

```
# 完整读取流程
WRITE_ENABLE → PAGE_PROGRAM → READ_STATUS(等待BUSY=0)

# 完整擦除流程
WRITE_ENABLE → SECTOR_ERASE → READ_STATUS(等待BUSY=0) → READ_DATA(验证0xFF)

# 进入QSPI模式流程
WRITE_ENABLE → WRITE_STATUS_REG(QE=1) → FAST_READ_QUAD_IO
```
