## 概述

本文档是面向高级/资深嵌入式工程师的FreeRTOS面试复习资料大纲。与基础面试资料不同，本大纲侧重于：

1. **源码实现深度分析** - 不仅回答"是什么"，更要回答"怎么做"和"为什么这样实现"
2. **工程实践问题** - 实际项目中遇到的疑难问题排查与解决
3. **系统设计能力** - 从架构角度分析RTOS系统设计
4. **性能调优** - 实时性、内存、功耗的深度优化

### 现有资料 vs 本大纲

| 维度 | 现有基础资料 | 本大纲(高级) |
|------|-------------|-------------|
| 调度算法 | 概念介绍 | 优先级位图O(1)实现细节、汇编代码分析 |
| 任务通信 | API使用 | 队列内部实现、零拷贝优化 |
| 内存管理 | heap_1~5区别 | 内存分配器源码分析、碎片处理 |
| 中断处理 | FromISR API | 中断嵌套、优先级配置、延迟处理 |
| 实时性 | 概念介绍 | WCRT计算、抖动分析 |
| 多核 | SMP概念 | 核间同步、缓存一致性 |
| 调试 | Tracealyzer使用 | 问题定位、源码级调试 |
