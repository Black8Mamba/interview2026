# FreeRTOS 学习文档设计方案

## 文档目标

为面试准备和系统学习 FreeRTOS，建立从基础到高级的完整知识体系。

- 面向人群：各level开发者，重点覆盖初/中/高级内容
- 文档风格：理论与代码交叉讲解，面试导向
- 目标平台：STM32（ARM Cortex-M 系列）

## 文档架构

### 目录结构

```
E:/ai/freertos/docs/
├── index.md              ← 主入口（学习路径总览 + 快速跳转）
├── 01_overview.md        ← FreeRTOS 概述与架构
├── 02_configuration.md   ← 配置项详解
├── 03_task_management.md ← 任务管理
├── 04_scheduling.md       ← 任务调度与切换
├── 05_memory_management.md ← 内存管理
├── 06_synchronization.md  ← 同步机制
├── 07_interrupt.md        ← 中断管理与定时器
├── 08_porting.md          ← 移植适配
└── 09_debugging.md        ← 调试方案
```

### 每章统一模板

1. **原理讲解** — 核心概念、数据结构、设计思想
2. **代码示例** — STM32F4/F1 实战代码，逐行解读
3. **面试重点** — 高频问题清单 + 参考答案要点
4. **避坑指南** — 常见错误与最佳实践

每个章节完全独立，读者可按需跳读，不依赖前后章节，适合面试快速查阅。
