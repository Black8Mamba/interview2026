## 使用方法

1. **第一阶段**：阅读知识大纲，建立知识体系
   - 任务管理 → 调度机制 → 内存管理 → 同步通信 → 中断管理 → 功耗管理

2. **第二阶段**：通过面试题检验掌握程度
   - 基础概念 (30题) → 原理深度 (50题) → 实际应用 (70题) → 进阶优化 (50题)

3. **第三阶段**：阅读项目案例，展示实际经验
   - 智能家居网关（消费电子）
   - 车载信息娱乐系统（汽车电子）

4. **第四阶段**：面试前使用 Checklist 快速复习
